# Employee Management App

A responsive Angular mini-project for employee management with:
- Login page with hard-coded functional authentication
- Employee list with 105 dummy records, search, sorting, paging, and per-page selection
- Add employee form with mandatory validation, email validation, date validation, and searchable group dropdown
- Employee detail page with formatted salary display

## Requirements
- Node.js 18+
- npm 9+

## Run locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Open the app at http://localhost:4200
